# Kits

Kits is a library for functional programming in Scala.

[![Build Status](https://travis-ci.org/halcat0x15a/kits.svg)](https://travis-ci.org/halcat0x15a/kits)

## Getting Started

```scala
libraryDependencies += "org.halcat" %% "kits" % "0.4.0"
```
